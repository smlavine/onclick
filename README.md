# [onclick](https://sr.ht/~smlavine/onclick)

```
Usage: ./onclick [-hC] command [arguments...]
```

Every time the terminal is clicked, or enter is pressed, the command runs.

## Installation

To build, run `make`.

To install, run `make install`.

To uninstall, run `make uninstall`.
